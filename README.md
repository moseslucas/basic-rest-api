# Basic REST API

Just 1 Model with 1 Field
To testout Apollo API Integrating REST API to any frontend

URL: 
```
https://rest-api-link-apollo.herokuapp.com
```

### GET Games
Endpoint
```
https://rest-api-link-apollo.herokuapp.com/games
```

